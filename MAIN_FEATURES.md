# 主な機能（ドラフト）

## ユーザー用アプリ
- 各店舗が発行するスタンプを貯め、10個達成でその店舗の指定クーポンが利用可能。
  - 現状は1枚のスタンプカードで最大10個まで。
  - 将来的に2枚目以降のスタンプカードを発行する可能性あり。
- 下部タブは「ホーム / マップ / 投稿 / アカウント」で構成。
  - 中央にフローティングボタンとしてQRコードを配置。
- ホーム画面で以下を表示してゲーミフィケーションを促進。
  - 保有スタンプ合計数
  - 新規開拓店舗数（スタンプを1個以上獲得した店舗）
  - オールコンプリート店舗数（スタンプを10個達成した店舗）
- 条件抽出によるクーポン表示・取得。
  - 来店済みで未使用のクーポン
  - 近くのお店のクーポン
  - あと少しで達成できそうな店舗のクーポン
- 条件抽出による店舗投稿の一部表示。
- マップ画面（Google Maps）
  - 加盟店のピンを強調表示。
  - 非加盟店のピンは目立たない/非表示。
  - モード切替: 「新規開拓」「営業中」「ジャンル」など。
- ピンタップで店舗の簡易情報をポップアップ表示。
  - 店舗アイコン、営業時間、スタンプ数、ジャンル、詳細ボタン
  - 詳細ボタンで店舗詳細画面へ遷移
- QRコード（店舗用アプリで読み取り）
  - 店舗で使えるクーポン一覧を表示。
  - クーポンにチェックを入れて「次へ」で、ユーザー情報を表示（保有スタンプ数、来店回数、ユーザー名、アイコンなど）。
  - 「スタンプを押印する」でユーザー側のスタンプを1つ加算。
  - 押印画面はアニメーション付きでゲーミフィケーション要素を付与。
- 投稿タブ
  - 各店舗の投稿をSNSのように閲覧可能。
  - 投稿をタップすると詳細画面へ遷移。
  - 画像は1:1で表示、下にタイトル/本文。
  - コメント機能・いいね機能を提供。
- アカウントタブ
  - 各種設定、プロフィール編集、通知設定が可能。
