## ぐるまっぷとは？

**"知らなかった小さな名店"に出会う地図** をコンセプトにした、中小飲食店のための集客プラットフォームアプリです。

- マップで近くのお店を**自然に発見**してもらえます
- デジタルスタンプカードで**リピーター**を育てます
- データ分析で**集客の効果が見える**ようになります

| 項目 | 内容 |
|------|------|
| 対応端末 | iPhone / Android |
| 店舗アプリ | 専用の店舗管理アプリを無料提供 |
| 初期費用 | **0円** |
| 契約期間 | **縛りなし**（いつでも解約可能） |

---

## お店が得られるメリット

### 1. 新しいお客さんに見つけてもらえる

- ユーザーがアプリを開くと、近くのお店が**マップ上に表示**されます
- まだ行ったことがないお店を**自動でおすすめ**する機能があります
- 他の加盟店のお客さんが、あなたのお店を発見 → **新規来店**に繋がります
- 加盟店が増えるほど、**送客の効果も大きくなります**

### 2. 低コスト・低リスクで始められる

- 初期費用**0円**、契約縛り**なし**
- いつでも解約可能なので、**リスクゼロ**で試せます
- 先着5店舗まで全機能を**無料**で利用可能

### 3. 日常の操作はNFCタッチ or QRスキャンだけ

- **NFCタッチ**: お客さんがスマホをかざすだけ。スタッフの操作は**不要**
- **QRスキャン**: お客さんのQRコードをスキャンするだけ。**約10秒**で完了
- スマホが苦手でも**すぐに使える**シンプルな操作
- 初期設定はサポートが**一緒にお手伝い**します

## 機能のご紹介

### スタンプカード

| 項目 | 内容 |
|------|------|
| 仕組み | お客さんが来店 → NFCタッチ or QRスキャン → スタンプ1個獲得 |
| スタンプ数 | **10個**で1枚のカード達成（累積カウント・リセットなし） |
| 達成特典 | **値引き型のみ**（例: 「100円引き」「500円引き」「会計から10%引き」など）。お店が自由に金額を設定 |
| 2枚目以降 | スタンプは**リセットなし**で累積。10個達成ごとに「2枚目→3枚目…」と次のカードへ自動進行 |
| クーポン発行 | **何個目でクーポンを発行するかは各店舗が自由に設定**（店舗裁量）。例: 3個目・5個目・7個目で発行する店舗もあれば、9個目・10個目で発行する店舗もある。設定した必要スタンプ数に達すると**自動付与**（有効期限なし・累積保有可） |
| 紙のカード | **不要**。スマホで完結するので忘れる・なくす心配なし |
| 物理カードの移行 | 既存の紙スタンプカードを保有しているお客さんのスタンプを引き継げます（下記「物理スタンプカード移行機能」参照） |

### 物理スタンプカード移行機能

紙のスタンプカードをご利用のお客さんをスムーズにアプリへ移行できます。

**操作手順（店舗スタッフ）**:
1. 設定画面 →「物理スタンプカード移行」をタップ
2. お客さんのアプリのQRコードをスキャン
3. 紙カードのスタンプ数を**目視確認**して入力（1〜99）
4. 移行後のスタンプ数を確認して「移行を実行する」→「確認済み・移行する」

**ルール**:
- 1人のお客さんにつき1店舗あたり1回のみ実行可能（二重移行防止）
- 来店ボーナスコインは付与されません（実来店ではないため）
- スタンプが10の倍数に達した場合、スタンプ達成特典クーポンが自動付与されます
- 物理カードは必ず目視確認した上で移行してください

### NFCタッチでセルフチェックイン

レジ横や入口に設置された**NFCタグ**にお客さんがスマホをかざすだけで、来店記録が完了します。

| 項目 | 内容 |
|------|------|
| 操作 | お客さんがスマホをNFCタグにかざすだけ（**スタッフ操作不要**） |
| スタンプ | 自動で1個付与（1店舗1日1回まで） |
| クーポン利用 | チェックイン時に使えるクーポンを選択可能。利用後は**6桁認証コード + リアルタイム時計**を画面表示 |
| スタッフ確認 | お客さんの画面に表示された認証コードと時計を目視確認するだけでOK |
| 設置物 | NFCタグシール付きスタンディングカード（運営から提供） |

> **NFCタグは運営がご用意します。**設置はレジ横に置くだけ。電源も充電も不要です。

### クーポン

| 項目 | 内容 |
|------|------|
| 発行数 | 1店舗あたり**最大3枚**まで同時発行 |
| クーポンの種類 | 割引クーポン / プレゼントクーポン / 特別オファー の3タイプ（スタンプ達成特典は割引クーポンのみ） |
| 有効期限 | **1日〜2週間**の範囲で自由に設定 |
| 発行枚数 | 枚数制限あり、または**無制限**を選択可能 |
| 通知 | フォロー中のお客さんに**プッシュ通知**でお知らせ |
| 効果測定 | クーポン別の利用数・推移を**グラフで確認** |

#### 会計時のクーポン確認（店舗スタッフ向け）

お客さんのQRコードをスキャンすると、クーポン選択画面に以下の2種類が表示されます。

| クーポン種別 | 内容 | 表示条件 |
|-------------|------|---------|
| **通常クーポン** | お店が独自発行したクーポン | 有効期限内のもの |
| **コイン交換クーポン** | コイン10枚で取得した100円引き（ミッション経由） | 未使用・有効期限内のもの |
| **スタンプ達成クーポン** | 店舗が設定した必要スタンプ数に達すると自動付与されるクーポン | 未使用のもの（有効期限なし） |

スタッフは会計時にこれらをまとめて確認でき、お客さんが希望するクーポンを選択して使用処理を完了できます。

### フォロー＋プッシュ通知

お客さんが来店してスタンプを獲得すると、**自動的にお店をフォロー**します。

- お店がクーポンを発行 → フォロー中のお客さんに**通知**
- お店が投稿を作成 → フォロー中のお客さんに**通知**
- **Instagram連携で投稿を同期** → フォロー中のお客さんに**通知**
- お客さんは通知を見て「あ、あのお店だ」→ **再来店**

> 1回来たお客さんとの接点を、自然に維持する仕組みです

## 送客ネットワーク

ぐるまっぷの最大の特徴は、**加盟店同士でお客さんを送り合う**仕組みです。

```
  A店のお客さん → マップで B店を発見 → B店に初来店
  B店のお客さん → マップで A店を発見 → A店に初来店
```

- マップ上で未訪問の店舗が**自動的におすすめ**されます
- あなたのお客さんが他のお店を発見し、他のお店のお客さんがあなたを発見します
- **加盟店が増えるほど、送客の効果も大きくなります**

> 競合ではなく、**お互いにお客さんを送り合う**エコシステムです

## お客さんが使うゲーム機能

ぐるまっぷでは、お客さんが「つい開きたくなる」ゲーム要素を搭載しています。
これらはすべて**お店側の手間なし**で自動的に動きます。

### 初回マップ体験ミッション（お客さんのアプリDL促進）

新規登録後のチュートリアル（4ページスライド）完了後、**インタラクティブウォークスルー**が自動で始まります。画面にガイドが表示され、お客さんは指示に従って操作するだけで自然にアプリの使い方を覚えられます。

1. お客さんがアプリを登録 → チュートリアル4ページを確認
2. ウォークスルーが自動開始：「マップ画面を開いてみよう！」→ マップタブをタップ
3. 「気になるお店をタップしてみよう！」→ マップ上の店舗ピンを1つタップ
4. 店舗情報パネルを確認 → ホーム画面に戻る
5. 「ミッションを確認してコインを受け取ろう！」→ ミッションボタンをタップ → **10コイン即獲得**
6. 「10コインで100円引きクーポンと交換しよう！」→ ホーム画面のコイン交換ボタンをタップ
7. 10コインで**100円引きクーポン**に交換 → その場で利用可能

> **店舗スタッフの案内**: 「アプリを入れていただくと、今日のお会計が100円引きになりますよ」
> この一言でOKです。100円引きクーポンの原資はプラットフォーム負担のため、お店の費用は一切かかりません。

### バッジ（全162種）

来店やスタンプ達成などの条件を満たすとバッジを獲得。コレクション欲で継続利用を促進します。

| バッジの例 | 獲得条件 |
|-----------|---------|
| スタンプ系 | スタンプ合計○個達成（15段階） |
| 来店系 | ○回来店達成 |
| カテゴリ系 | カフェ/ラーメン/居酒屋 等のジャンル別来店（10グループ×5段階）|
| アクション系 | お気に入り登録、クーポン使用、コメント投稿 など |

- 公平性維持のため、マップ閲覧に紐づく報酬カウントは1日1回のみ加算されます。

### コインシステム

アプリを使うことで「コイン」が貯まり、クーポンと交換できます。

| 獲得方法 | コイン数 |
|---------|---------|
| **初回マップ体験ミッション** | **+10コイン（初回のみ）** |
| デイリーミッション（3種） | 毎日最大3コイン |
| 3日連続ログイン | +2コイン |
| 7日連続ログイン | +5コイン |
| 30日連続ログイン | +10コイン |
| 来店（スタンプ獲得時） | +1コイン |
| 友達紹介（初回スタンプ獲得時） | +設定コイン数（デフォルト5コイン） |

**コインの使い道**: 10コインで未訪問店舗の**100円引きクーポン**に交換
→ あなたのお店にまだ来たことがないお客さんが、このクーポンをきっかけに初来店する可能性があります
→ 交換候補に表示される店舗は、公開中（`isActive=true`）かつ承認済み（`isApproved=true`）かつ `stores.isOwner!=true` の条件を満たす店舗です

> コインの有効期限は最終獲得日から180日です
> コイン交換クーポン（100円引き）の有効期限は取得から30日間です。有効期限の7日前・3日前に自動でお知らせ通知が届くため、使い忘れを防止できます

## 店舗用アプリの操作

### 日常の操作（毎日やること）

| 操作 | やり方 | 所要時間 |
|------|-------|---------|
| スタンプ付与（NFCタッチ） | お客さんがレジ横のNFCタグにスマホをかざす → **自動完了** | **約1秒**（スタッフ操作不要） |
| スタンプ付与（QRスキャン） | お客さんのQRコードをスキャン → 確認 → 完了 | **約10秒** |

NFCタグを設置済みの場合、日常のスタンプ付与は**お客さんのセルフ操作のみ**で完結します。QRスキャン方式も引き続きご利用いただけます。

### 必要に応じてやること

| 操作 | 内容 |
|------|------|
| クーポン作成 | クーポン管理画面の下部固定「新規クーポンを作成」ボタンから、タイトル・タイプ・有効期限・画像を入力して発行 |
| ニュース作成 | 画像・タイトル・本文・掲載期間を入力して配信 |
| メニュー更新 | メニューの追加・編集・並び替え |
| データ確認 | 分析ダッシュボードで来店者数やリピート率を確認 |
| 投稿作成 | 投稿管理画面の下部固定「新規投稿を作成」ボタンから、写真+テキストで投稿（プレミアム） |

## 始め方

### STEP 1: アプリをダウンロード（1分）

App Store または Google Play で「ぐるまっぷ 店舗」を検索してダウンロード

### STEP 2: アカウント登録（5分）

- メールアドレスとパスワードを入力
- メール認証（6桁のコードを入力）

### STEP 3: 店舗紐づけ（1分）

店舗情報はこちらで事前に登録しますので、お店側の入力作業はありません。

- アプリにログインすると紐づけ画面が表示されます
- お店を選択すると6桁の確認コードが表示されます
- 確認コードをこちらにお伝えください
- こちらで確認完了後、すぐに利用開始できます

### STEP 4: 利用開始

紐づけ完了後、以下を設定すると準備完了です。
- スタンプ達成特典の設定
- メニューの登録
- 店内画像のアップロード
- 決済方法の設定
- 臨時休業・特別営業日の登録（設定 → 営業カレンダー）※ 登録内容はユーザー側の店舗詳細「今週の営業時間」と マップ「営業中のみ」フィルターにリアルタイム反映

> 設定はサポートが一緒にお手伝いしますので、お気軽にご相談ください

## よくあるご質問

### Q. 初期費用はかかりますか？
**A.** いいえ、初期費用は**0円**です。月額料金のみでご利用いただけます。

### Q. 契約期間の縛りはありますか？
**A.** ありません。**いつでも解約可能**です。

### Q. スマホが苦手でも大丈夫ですか？
**A.** はい。日常の操作は**QRコードをスキャンするだけ**です。初期設定はサポートが一緒にお手伝いします。

### Q. 常連さんが多いのですが、新規客も必要ですか？
**A.** ぐるまっぷでは、常連さんが他のお店を回遊する際に、**別のお店の常連さんがあなたのお店を発見**します。常連さんを大切にしながら、自然と新規のお客さんも増えていきます。

### Q. 大手の掲載サービスとの違いは何ですか？
**A.** 大手は「検索して探す」仕組みです。ぐるまっぷは**「マップ上で自然に見つかる」**仕組みなので、お客さんが知らなかったお店に出会えます。また、**来店数やリピート率**まで測れるのが大きな違いです。

### Q. 効果が出なかった場合は？
**A.** 契約縛りがないので、いつでも解約できます。ただ、月次レポートで**一緒にデータを見て改善策を考えます**ので、まずはご相談ください。

### Q. 複数店舗を登録できますか？
**A.** はい。1つのアカウントで**複数店舗の管理**が可能です。店舗の切り替えもアプリ内で簡単にできます。

### Q. クーポンの費用は誰が負担しますか？
**A.** スタンプ達成特典やクーポンの費用は**お店のご負担**となります。スタンプ達成特典は**値引き型のみ**（例: 100円引き、10%引き）で、お店が金額・割引率を設定できます。なお、現在はコイン交換による初来店クーポン（100円引き）の原資を**プラットフォームが負担**しています。
