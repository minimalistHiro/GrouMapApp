## ぐるまっぷとは？

**"知らなかった小さな名店"に出会う地図** をコンセプトにした、中小飲食店のための集客プラットフォームアプリです。

- マップで近くのお店を**自然に発見**してもらえます
- デジタルスタンプカードで**リピーター**を育てます
- データ分析で**集客の効果が見える**ようになります

| 項目 | 内容 |
|------|------|
| 対応端末 | iPhone / Android |
| 店舗アプリ | 専用の店舗管理アプリを無料提供 |
| 初期費用 | **0円** |
| 契約期間 | **縛りなし**（いつでも解約可能） |

---

## お店が得られるメリット

### 1. 新しいお客さんに見つけてもらえる

- ユーザーがアプリを開くと、近くのお店が**マップ上に表示**されます
- まだ行ったことがないお店を**自動でおすすめ**する機能があります
- 他の加盟店のお客さんが、あなたのお店を発見 → **新規来店**に繋がります

### 2. 来てくれたお客さんがリピーターになる

- 来店時にQRコードをスキャンするだけで**スタンプ**が貯まります
- スタンプが貯まるほど「あと少しで特典！」と**再来店の動機**が強まります
- 来店したお客さんは自動的にお店を**フォロー**し、新しいクーポンや投稿があると**プッシュ通知**が届きます

### 3. 集客の効果が数字で見える

- 新規顧客数・リピート率・クーポン利用率を**リアルタイムで確認**できます
- 月に1回、担当者が**一緒にデータを確認**して改善策を提案します
- 「掲載して終わり」ではなく、**一緒に集客を改善するパートナー**です

## 機能のご紹介

### スタンプカード

| 項目 | 内容 |
|------|------|
| 仕組み | お客さんが来店 → QRスキャン → スタンプ1個獲得 |
| スタンプ数 | **10個**で1枚のカード達成 |
| 達成特典 | **お店が自由に設定**（例: ドリンク1杯無料、100円引き など） |
| 2枚目以降 | 達成後も自動で次のカードを発行 |
| 紙のカード | **不要**。スマホで完結するので忘れる・なくす心配なし |

### クーポン

| 項目 | 内容 |
|------|------|
| 発行数 | 1店舗あたり**最大3枚**まで同時発行 |
| クーポンの種類 | 割引 / 無料 / 値引き / キャッシュバック の4タイプ |
| 有効期限 | **1日〜2週間**の範囲で自由に設定 |
| 発行枚数 | 枚数制限あり、または**無制限**を選択可能 |
| 通知 | フォロー中のお客さんに**プッシュ通知**でお知らせ |
| 効果測定 | クーポン別の利用数・推移を**グラフで確認** |

### フォロー＋プッシュ通知

お客さんが来店してスタンプを獲得すると、**自動的にお店をフォロー**します。

- お店がクーポンを発行 → フォロー中のお客さんに**通知**
- お店が投稿を作成 → フォロー中のお客さんに**通知**
- お客さんは通知を見て「あ、あのお店だ」→ **再来店**

> 1回来たお客さんとの接点を、自然に維持する仕組みです

### ニュース配信

|------|------|
| 内容 | お知らせ、キャンペーン、季節メニュー など |
| 画像 | 正方形（1:1）の画像を設定 |
| 掲載期間 | 開始日〜終了日を指定して**自動で公開/非公開** |
| 表示場所 | ユーザーアプリのホーム画面に横スクロールで表示（最大7件）|

### メニュー管理

| 項目 | 内容 |

## 送客ネットワーク

ぐるまっぷの最大の特徴は、**加盟店同士でお客さんを送り合う**仕組みです。

```
  A店のお客さん → マップで B店を発見 → B店に初来店
  B店のお客さん → マップで A店を発見 → A店に初来店
```

- マップ上で未訪問の店舗が**自動的におすすめ**されます
- あなたのお客さんが他のお店を発見し、他のお店のお客さんがあなたを発見します
- **加盟店が増えるほど、送客の効果も大きくなります**

> 競合ではなく、**お互いにお客さんを送り合う**エコシステムです

## お客さんが使うゲーム機能

ぐるまっぷでは、お客さんが「つい開きたくなる」ゲーム要素を搭載しています。
これらはすべて**お店側の手間なし**で自動的に動きます。

### バッジ（全141種）

来店やスタンプ達成などの条件を満たすとバッジを獲得。コレクション欲で継続利用を促進します。

| バッジの例 | 獲得条件 |
|-----------|---------|
| スタンプ系 | スタンプ合計○個達成（15段階） |
| 来店系 | ○回来店達成 |
| カテゴリ系 | カフェ/ラーメン/居酒屋 等のジャンル別来店（10グループ×5段階）|
| アクション系 | お気に入り登録、クーポン使用、コメント投稿 など |

### コインシステム

アプリを使うことで「コイン」が貯まり、クーポンと交換できます。

| 獲得方法 | コイン数 |
|---------|---------|
| デイリーミッション（3種） | 毎日最大3コイン |
| 3日連続ログイン | +2コイン |
| 7日連続ログイン | +5コイン |
| 30日連続ログイン | +10コイン |
| 来店（スタンプ獲得時） | +1コイン |

**コインの使い道**: 10コインで未訪問店舗の**100円引きクーポン**に交換
→ あなたのお店にまだ来たことがないお客さんが、このクーポンをきっかけに初来店する可能性があります

### ランキング

コイン数・バッジ数・スタンプ数で全ユーザーのランキングを表示。競争心で利用を促進します。

## 店舗用アプリの操作

| 操作 | やり方 | 所要時間 |
|------|-------|---------|

## 始め方

### STEP 1: アプリをダウンロード（1分）

App Store または Google Play で「ぐるまっぷ 店舗」を検索してダウンロード

### STEP 2: アカウント登録（5分）

- メールアドレスとパスワードを入力
- メール認証（6桁のコードを入力）

### STEP 3: 店舗情報を入力（10〜15分）

| 入力項目 | 内容 |
|---------|------|
| 店舗名 | お店の名前 |
| 経営形態 | 個人 / 法人 |
| 住所 | お店の住所 |
| カテゴリ | ジャンル（カフェ、ラーメン、居酒屋 など） |
| 営業時間 | 曜日ごとの営業時間 |
| 店舗画像 | スマホで撮影したものでOK |
| 設備・サービス | 駐車場/テイクアウト/Wi-Fi 等の対応状況 |

### STEP 4: 承認を待つ（最短当日）

運営チームが店舗情報を確認し、承認します。

### STEP 5: 利用開始

承認後、以下を設定すると準備完了です。
- スタンプ達成特典の設定
- メニューの登録
- 店内画像のアップロード
- 決済方法の設定

> 設定はサポートが一緒にお手伝いしますので、お気軽にご相談ください

## よくあるご質問

### Q. 初期費用はかかりますか？
**A.** いいえ、初期費用は**0円**です。月額料金のみでご利用いただけます。

### Q. 契約期間の縛りはありますか？
**A.** ありません。**いつでも解約可能**です。

### Q. スマホが苦手でも大丈夫ですか？
**A.** はい。日常の操作は**QRコードをスキャンするだけ**です。初期設定はサポートが一緒にお手伝いします。

### Q. 常連さんが多いのですが、新規客も必要ですか？
**A.** ぐるまっぷでは、常連さんが他のお店を回遊する際に、**別のお店の常連さんがあなたのお店を発見**します。常連さんを大切にしながら、自然と新規のお客さんも増えていきます。

### Q. 大手の掲載サービスとの違いは何ですか？
**A.** 大手は「検索して探す」仕組みです。ぐるまっぷは**「マップ上で自然に見つかる」**仕組みなので、お客さんが知らなかったお店に出会えます。また、**来店数やリピート率**まで測れるのが大きな違いです。

### Q. 効果が出なかった場合は？
**A.** 契約縛りがないので、いつでも解約できます。ただ、月次レポートで**一緒にデータを見て改善策を考えます**ので、まずはご相談ください。

### Q. 複数店舗を登録できますか？
**A.** はい。1つのアカウントで**複数店舗の管理**が可能です。店舗の切り替えもアプリ内で簡単にできます。

### Q. クーポンの費用は誰が負担しますか？
**A.** スタンプ達成特典やクーポンの費用は**お店のご負担**となります。特典の内容・金額はお店が自由に設定できます。なお、現在はコイン交換による初来店クーポン（100円引き）の原資を**プラットフォームが負担**しています。
