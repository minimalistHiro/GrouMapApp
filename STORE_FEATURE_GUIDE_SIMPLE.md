# ぐるまっぷ 店舗オーナー様向けかんたんガイド

> 「知らなかった小さな名店」に出会う地図アプリ — 中小飲食店のための集客プラットフォーム

---

## 概要

| 項目 | 内容 |
|------|------|
| 対応端末 | iPhone / Android |
| 初期費用 | **0円** |
| 契約期間 | **縛りなし**（いつでも解約可） |

---

## 3つのメリット

1. **新しいお客さんに発見される** — マップ+レコメンドで自動表示。他店のお客さんがあなたのお店を自然に発見し、新規来店に繋がる
2. **低コスト・低リスクで始められる** — 初期費用0円、契約縛りなし。いつでも解約できるのでリスクゼロ
3. **日常の操作はNFCタッチ or QRスキャンだけ** — NFCタグにかざすだけ（スタッフ操作不要）or QRスキャン（約10秒）。スマホが苦手でもすぐ使える

## 主な機能

### 図鑑発見コレクション（NFCタッチ方式）
- 来店 → NFCタッチ → **図鑑カードを発見**（ゲームのような体験）
- 初来店は暗転＋グロー光＋カードフリップ「発見！」演出でゲームのような非日常体験を提供
- 全店舗が「図鑑」として一覧表示。未発見店舗は「?」マーク表示でコレクション欲を刺激
- レア度（コモン〜レジェンド）が発見者数に応じて自動設定される
- **お店の手間はゼロ**。NFCタグを置くだけで自動で機能する

### スタンプカード（既存ユーザー向け救済措置として継続）
- 既存スタンプ保有者は引き続きNFCタッチ or QRスキャンでスタンプ加算継続
- スタンプは累積・リセットなし。紙のカード不要
- 既存の紙カードからの移行機能あり
- **新規ユーザーは図鑑発見方式に完全移行**（スタンプ付与なし）

### クーポン
- 最大**3枚**同時発行（有効期限: 1日〜2週間）
- 割引 / プレゼント / 特別オファーの3タイプ
- フォロワーにプッシュ通知でお知らせ

### フォロー＋通知
- 来店したお客さんが**自動フォロー**
- クーポン発行・投稿作成・Instagram同期時にプッシュ通知で再来店を促進

### NFCセルフチェックイン
- レジ横のNFCタグにお客さんがスマホをかざすだけで来店記録完了（**スタッフ操作不要**）
- クーポン利用も画面提示+6桁認証コードの目視確認でOK
- NFCタグは運営が提供。電源不要・置くだけ
- **予備タグを同梱**。剥がれ・破損時はお店で交換可能
- タグ盗難時はGrouMapに連絡するだけで旧タグを即無効化（新URL発行・お店の操作不要）

### その他の機能
- **ニュース配信** — お知らせ・キャンペーンをホーム画面に掲載
- **メニュー管理** — カテゴリ別に写真・価格・オプション付きで表示
- **投稿・Instagram連携** — 写真+テキスト投稿、Instagram自動同期（プレミアム）
- **分析ダッシュボード** — 来店推移・リピート率・クーポン効果・送客データ

## 送客ネットワーク

加盟店同士でお客さんを送り合う仕組みです。

```
A店のお客さん → マップでB店を発見 → B店に初来店
B店のお客さん → マップでA店を発見 → A店に初来店
```

**加盟店が増えるほど、送客効果も大きくなります。**

---

## ユーザー向けゲーム機能（お店の手間なし）

| 機能 | 内容 |
|------|------|
| **図鑑発見コレクション** | NFCタッチで「発見！」演出。全店舗を図鑑スタイルで一覧表示。未発見店舗は「?」マーク表示でコレクション欲を刺激 |
| **レア度システム** | 発見者数が少ない店舗ほどレア度が高い（レジェンド〜コモン）。「あなたが最初の発見者です！」などの特別メッセージ |
| **初回ウォークスルー** | チュートリアル後にインタラクティブウォークスルー（4ステップ）→ マップ操作を自然に体験 |
| **バッジ（192種）** | NFCチェックイン・発見達成などでバッジ獲得。コレクション欲で継続利用 |

> 100円引き特典はプラットフォームが後日現金精算（店舗の費用ゼロ）。
> スタッフの案内: 「アプリを入れると今日100円引きになりますよ」でOK。

## 日常の操作

| 操作 | 所要時間 |
|------|---------|
| スタンプ付与（NFCタッチ） | **約1秒**（スタッフ操作不要） |
| スタンプ付与（QRスキャン） | **約10秒** |

NFCタグ設置済みなら日常のスタッフ操作はゼロ。クーポン発行・データ確認などは必要に応じて。

## 始め方（4ステップ）

1. **アプリDL**（1分）— 「ぐるまっぷ 店舗」で検索
2. **アカウント登録**（5分）— メール+パスワード+認証コード
3. **店舗紐づけ**（1分）— こちらからお伝えする**6桁のリンクコード**を入力→即座に紐づけ完了
4. **利用開始** — スタンプ特典・メニュー・店内画像を設定

> 店舗情報はこちらで事前に登録します。設定はサポートが一緒にお手伝いします。
